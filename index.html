API Basejump: Timestamp microservice for FCC

User stories:
    1) I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: August 1, 2016)
    2) If it does, it returns both the Unix timestamp and the natural language form of that date.
    3) If it does not contain a date or Unix timestamp, it returns null for those properties.
Bonus: Get the Timestamp for the present time https://timestamp-serv.herokuapp.com/now

Example usage:

https://timestamp-serv.herokuapp.com/April%2028,%202016
https://timestamp-serv.herokuapp.com/1450137600

Example output:

{"unix":1461801600,"natural":"April 28, 2016"}
Source Code is available at https://github.com/ChaituVR/timestamp-service
